Diaosi
======
A dictionary made by a diaosi, for the diaosi

Feature
=======
1. up to 100,000 words int plain text file
2. quick loading and searching
3. spelling correction
4. cross platform

Install
=======

1. install sdict module
        
        
        pip install sdict
    
        or
    
        easy_install sdict
        
2. git clone https://github.com/fxsjy/diaosi

3. python ds.pyw
